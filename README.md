# GitHub Copilot Playground

Welcome to GitHub Copilot Playground — a collection of projects, scenarios, and guided experiments designed to help you explore what’s possible with GitHub Copilot and GitHub Copilot CLI!

This repository is a place to test workflows, try prompts, and experiment with real-world development tasks without starting from a blank slate. Whether you're generating commit messages, debugging issues, creating pull requests, cleaning up code, exploring terminal workflows, or testing new Copilot features, each project is designed to give you something practical to interact with.

Inside you'll find intentionally structured projects and demo scenarios that create opportunities to explore, break things, and see how GitHub Copilot responds.

Think of this repo like a workshop space for experimenting with AI-assisted development.

Clone a project, open it up, and start exploring!

## Prerequisites

- Python 3.8+
- A GitHub Copilot [subscription](https://github.com/features/copilot/plans)
- Visual Studio Code
- [Python extension](https://marketplace.visualstudio.com/items?itemName=ms-python.python) for Visual Studio Code
